# Kasoku

[English](README.md)

開発ワークフローを加速させる、生産性向上機能を提供する VSCode 拡張機能です。

## 機能

### パスと行番号をコピー

コンテキストメニューからファイルパスと行番号を素早くクリップボードにコピーできます。

- **相対パスと行番号をコピー** - ワークスペースルートからの相対パスをコピー

  - 例: `src/index.ts:42`

- **絶対パスと行番号をコピー** - ファイルの絶対パスをコピー
  - 例: `/path/to/project/src/index.ts:42`

#### 使い方

1. VSCode で任意のファイルを開く
2. 参照したい行にカーソルを置く
3. 右クリックしてコンテキストメニューを開く
4. 以下のいずれかを選択：
   - 「相対パスと行番号をコピー」
   - 「絶対パスと行番号をコピー」

コマンドパレット（Cmd+Shift+P / Ctrl+Shift+P）からもアクセスできます。

## 多言語対応

この拡張機能は以下の言語をサポートしています：

- 英語
- 日本語

VSCode の言語設定に基づいて、UI の言語が自動的に選択されます。
